## 🚀 ¿Qué hace especial esta plataforma?

Este repositorio no es solo documentación — es una **WebApp de aprendizaje interactiva** que corre directo en tu navegador.

| Característica | Descripción |
|---|---|
| 💻 **Terminal Simulada** | Practica comandos Git en el navegador sin instalar nada |
| 📝 **Metodología Cornell** | Apuntes estructurados con claves, notas y resúmenes |
| 🎉 **Gamificación** | Sistema de progreso, confeti y certificación al 100% |
| 🔍 **Buscador Dinámico** | Localiza cualquier término técnico al instante |
| 🤖 **CI/CD Integrado** | GitHub Actions regenera el contenido en cada push |

## 🗺️ Temario Completo

<details>
<summary><b>📌 Módulo 1 — Arquitectura Profunda</b> (Nivel Core)</summary>

- Cómo funciona Git por dentro: Blobs, Trees y Commits
- Integridad criptográfica con SHA-256
- El modelo de objetos de Git
</details>

<details>
<summary><b>⚡ Módulo 2 — Flujo de Trabajo Profesional</b> (Nivel Pro)</summary>

- Instalación y configuración global
- Firma GPG y autenticación SSH
- Alias avanzados para productividad
- Ciclo de vida de archivos y Staging Area
- Commits Semánticos y Atómicos (Conventional Commits)
</details>

<details>
<summary><b>🌿 Módulo 3 — Gestión de Ramas y Conflictos</b> (Nivel Pro)</summary>

- Estrategia Merge vs Rebase — cuándo usar cada uno
- Cherry-pick y stash avanzado
- Resolución de conflictos como un profesional
</details>

<details>
<summary><b>🤝 Módulo 4 — Colaboración & Ecosistema GitHub</b> (Nivel Pro)</summary>

- Pull Requests y Code Review de nivel Senior
- Perfil de GitHub, Gists y Organizaciones
- GitHub Pages, Issues y Projects
</details>

<details>
<summary><b>🤖 Módulo 5 — Automatización & AI</b> (Nivel AI)</summary>

- GitHub Actions: CI/CD Pipelines desde cero
- Integración de LLMs en el flujo de trabajo Git
- Workflows avanzados con matrices y artefactos
</details>

<details>
<summary><b>⚡ Módulo 6 — Productividad & Rescate</b> (Nivel Senior)</summary>

- Git Worktrees para manejar ramas en paralelo
- Git Reflog: recuperar commits "perdidos"
- Git Bisect: encontrar bugs con búsqueda binaria
</details>

<details>
<summary><b>🏆 Módulo 7 — Maestría y Buenas Prácticas</b> (Nivel Senior)</summary>

- GitFlow vs Trunk Based Development
- Monorepos y estrategias de versionado semántico
- Proyecto Integrador Final
</details>
