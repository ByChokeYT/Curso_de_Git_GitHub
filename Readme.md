# 🦑 Git & GitHub Masterclass: Sistema de Control de Versiones Pro 🐙

![portada](https://github.com/ByChokeYT/Curso_de_Git_GitHub/blob/main/Recursos/Portada_Git_vs_Github.gif?raw=true)

> **De Cero a Senior:** Este no es un curso básico de Git. Es una inmersión profunda en la ingeniería de control de versiones, diseñada para desarrolladores que buscan dominar flujos de trabajo profesionales, automatización avanzada y resolución de crisis.

---

## 🚀 Plataforma de Aprendizaje "Elite"

Este repositorio no es solo código; es una **WebApp de aprendizaje interactiva** integrada.

- **💻 Terminal Simulada:** Practica comandos de Git directamente en el navegador con nuestra consola interactiva.
- **📝 Metodología Cornell:** Apuntes estructurados dinámicamente con sistema de claves, notas y resúmenes de ingeniería.
- **🎉 Gamificación:** Sistema de progreso con LocalStorage, efectos de confeti y certificación al completar el 100%.
- **🔍 Buscador con Resaltado:** Localiza términos técnicos al instante con nuestro motor de búsqueda dinámico.
- **🤖 CI/CD Ready:** Automatización total mediante GitHub Actions que regenera la base de datos del curso en cada push.
